
For building use
[source,shell]
----
sudo docker build --force-rm=true -t openfact-website:latest .
----

after successful build you can run the image by
[source,shell]
----
sudo docker run -d -p 80:80 openfact-website:latest
----

Or you can download and run the image without building it by
[source,shell]
----
sudo docker run -d -p 80:80 openfact/openfact-website:latest
----

This image is also in DockerHub under link:https://registry.hub.docker.com/u/openfact/openfact-website/[openfact/openfact-website:latest]
